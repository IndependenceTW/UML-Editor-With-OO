# UML-Editor-With-OO
For final Project, and more OO

## Basic Feature
- [ ] GUI of the UML Editor
- [ ] Object Creation
- [ ] Connection Line Creation
- [ ] Show the Port
- [ ] Object Selection
  - Single Object
  - Multi Objects
- [ ] Objects Group and Ungroup
- [ ] Move Objects
- [ ] Rename Objects

## Bonus Feature(More User-friendly)
- [ ] Show the Selection Range on Mouse Dragged
- [ ] Move the Object Animation
